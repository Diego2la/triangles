Program works only with console arguments.

Types of arguments:
	String   filename
	int 		number; 
	double   x1;
	double   y1;
	double   x2;
	double   y2;
	double   x3;
	double   y3;

Commands:
   
   // add new Triangle to file
   1) add <filename> <number>, <x1>, <y1>, <x2>, <y2>, <x3>, <y3>
